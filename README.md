# Hot Beans Web - Recruitment Portal
This project was developed for the **BTEC Level 3 Unit 6: Website Development** assessment.

## Project Overview
A responsive recruitment section for 'Hot Beans Web', designed to attract and recruit junior web developers. 

## Features
* **Company Profile:** Overview of Hot Beans Web.
* **Trainee Spotlights:** Real-world examples of our culture.
* **Job Specs:** Detailed requirements for junior roles.
* **Application Form:** Integrated form for prospective candidates.
* **Responsive Design:** Built using CSS Grid for mobile and desktop compatibility.

## Tech Stack
* HTML5
* CSS3 (Custom Dark Theme)
* GitHub Pages for Hosting

## Testing
Tested for functionality across Chrome, Firefox, and mobile Safari. All navigation links and external course links are verified.
